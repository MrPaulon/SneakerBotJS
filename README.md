# SneakerBotJS ☢️
Sneaker Bot Javascript.

Prérequis:
- NodeJS

## Run the bot

### 1.Installer les dépendances

```npm install```

### 2.Start le bot

```npm run startBot```

## NPM Packages

- puppeteer
- puppeteer-extra
- puppeteer-extra-plugin-stealth
- express

### (Dev dependencies)

- chalk
- nodemon
- dotenv
